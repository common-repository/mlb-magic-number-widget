== Description ==
Displays the current Magic for any Major League Baseball team as a sidebar widget.
You can place as many different reams as you want in the sidebar by dragging the widget.

== Installation ==

1. Download the plugin.
2. Upload the plugin to your wp-content/plugins directory.
3. Activate the plugin (all configuration is done through the widget).
4. Drag the widget to a sidebar.
5. Select a team.
6. Edit the title
10. Save
